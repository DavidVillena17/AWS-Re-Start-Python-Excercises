# AWS re/Start Python Exercises

¡Bienvenido a mi repositorio de prácticas de programación en Python! Este proyecto compila todos los ejercicios prácticos y proyectos que desarrollé durante el programa **AWS re/Start**, enfocados en construir una base sólida en desarrollo de software y automatización. Todo fue realizado con AWS Cloud9!

## 🚀 Acerca de este repositorio

Este repositorio refleja mi progreso y las habilidades prácticas adquiridas en la resolución de problemas lógicos y manipulación de datos utilizando Python. Cada script demuestra el dominio sobre conceptos fundamentales e intermedios de la programación, desde el manejo de variables hasta flujos de control e I/O de archivos.

## 🛠️ Habilidades Adquiridas y Aplicadas

A lo largo de estos módulos, he logrado consolidar las siguientes competencias técnicas:
- **Fundamentos de Python:** Dominio de los tipos de datos nativos, variables, operadores lógicos y matemáticos (`numeric-data.py`, `string-data-type.py`).
- **Estructuras de Datos Complejas:** Creación y manipulación estructurada de listas, diccionarios y tuplas para gestionar información escalable (`collectionss.py`, `composite-data.py`).
- **Control de Flujo:** Implementación robusta de bucles iterativos (`for-loop.py`, `while-loop.py`) y ramificación condicional para la toma de decisiones dinámicas (`conditionals.py`).
- **Archivos e I/O (Input/Output):** Lectura profunda y escritura de archivos locales de texto plano y hojas de cálculo CSV para almacenamiento persistente de datos (manipulación de `car_fleet.csv`).
- **Procesamiento y Limpieza de Datos (Bioinformática):** Uso avanzado de la librería `re` (Expresiones Regulares) para sanear cadenas de texto complejas, en este caso extrayendo y limpiando correctamente la secuencia proteínica de la preproinsulina humana para crear fragmentos individuales validables (`analyze-insuline.py`, `string-insulin.py`).

## 📂 Ejercicios Destacados

- **El Analizador de Insulina (`analyze-insuline.py`)** 
  Script que toma un archivo de texto en bruto extraído de una base biológica, elimina formatos no deseados mediante *Regex* e indexa los caracteres exactos que componen los enlaces de la insulina produciendo subarchivos limpios de secuencias de cadenas de caracteres.

- **Inventariado de Flota (`composite-data.py`)** 
  Procesa y transforma la lectura de un archivo `.csv` iterando a través de sus datos para convertirlos dinámicamente en diccionarios anidados en memoria (Deep Copying).
